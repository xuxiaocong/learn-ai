## 信息

* Python3.8

## 使用

```shell
git clone https://gitee.com/XuXiaoCong/learn-ai.git
cd book-handson-ml2
pip install -r requirements.txt
```

## 简介

* 本书分为上下两部分
    * 机器学习: Scikit-Learn、Numpy、Matplotlib、Pandas
    * 深度学习: Keras、TensorFlow2.0
